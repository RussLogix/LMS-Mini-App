# LMS-Mini-App
A simple Learning Management System (LMS) built with React, Vite, Node.js, and Express that allows users to view courses, browse lessons, and mark lessons as completed.

## Features

- View all courses
- View lessons for each course
- Mark lessons as completed (UI only)
--------------------------
## Setup

### Backend

1. cd backend
2. npm install
3. npm run dev

--------------------------

# Runs on:


http://localhost:5000

--------------------------

### Frontend

Open a new terminal:

1. cd frontend
2. npm install
3. npm run dev

# Runs on:

http://localhost:5173

--------------------------

## Tech Stack

- React (Vite)
- Node.js
- Express
- React Router DOM
